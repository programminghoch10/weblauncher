# weblauncher
Weblauncher is a startscreen launcher which displays a website, made for kiosk purposes.

__The no root version is currently beeing developed! Do not expect it to work.__

This version is made for not rooted devices! For better experience use the [root version](https://github.com/programminghoch10/weblauncher/tree/rootedversion) if possible.


## Features:
- 2 webpages switchable using volume keys
- reload webpage using volume keys
- always uses current webpage due to disabled cache
- auto enables wifi if disabled
- full screen
- acts as app launcher to be the only launchable application (kiosk mode)
- ~~pulls 2 website urls from build.prop~~ _(subject to change)_
  - ~~build.prop properties JJ.Page1 and JJ.Page2~~
  - if those arent defined, google.de and example.com are used
